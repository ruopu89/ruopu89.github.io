python总结_装饰器

發表於 2020-04-27 | 更新於 2020-11-04 | 分類於 python | 閱讀次數：

概念

  • 装饰器是对现有函数的增强
  • 装饰之后调用的还是原函数的名字，但功能已经是装饰后的效果了，调用原函数时结果是从装饰器返回的
  • 装饰器函数的内层函数就是原函数的增强代码，即实现原函数功能，又增加了功能
  • @符号是语法糖，函数加了@符号后，就会将其下面的函数当作参数传入到加了@符号的函数中
  • 带参装饰器可以有多个参数，并且也会把其下面的函数当作参数传入这个带参的函数中的内层函数里
  • 带参装饰器一般最多使用三层，每一层都返回其内层函数的函数名，这样就可以使用内层函数被上一层函数调用。只有最内一层的函数返回的是原函数的值
