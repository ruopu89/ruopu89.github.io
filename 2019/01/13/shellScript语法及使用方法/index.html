break、continue、exit对比

命令 说明
break n n表示跳出循环的层数，如果省略n表示跳出整个循环
continue n n表示退出到第n层继续循环，如果省略n表示跳过本次循环，忽略本次循环的接任代码，进入循环的下一次循环
exit n 退出当前shell程序，终止整个shell脚本，并返回n。n也可以省略。
return 与exit相似，在函数中使用，用于跳出函数
