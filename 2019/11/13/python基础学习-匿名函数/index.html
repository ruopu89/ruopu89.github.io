匿名函数

  • 匿名，即没有名字
  • 匿名函数，即没有名字的函数
    • 没有名字如何定义
    • 没有名字如何调用
    • 如果能调用，如何使用
  • python借助Lambda表达式构建匿名函数
  • 使用lambda关键字来定义匿名函数
  • 参数列表不需要小括号
  • 冒号是用来分割参数列表和表达式的
  • 不需要使用return，表达式的值，就是匿名函数返回值
  • lambda表达式(匿名函数)只能写在一行上，被称为单行函数
  • 用途
    • 在高阶函数传参时，使用lambda表达式，往往能简化代码
